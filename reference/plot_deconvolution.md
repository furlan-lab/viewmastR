# Plot deconvolution results

Plot deconvolution results

## Usage

``` r
plot_deconvolution(result, observed_counts = NULL, type = "all")
```

## Arguments

- result:

  Output from deconvolve_bulk()

- observed_counts:

  Original bulk counts matrix (for fit plots)

- type:

  Type of plot: "proportions", "fit", "residuals", or "all"
