# Package index

## viewmastR functions

Main functions used in wiewmastR

- [`viewmastR()`](https://furlan-lab.github.io/viewmastR/reference/viewmastR.md)
  : Integrate and Train Models on Reference Dataset and (Optionally)
  Infer on Query Datasets
- [`viewmastR_infer()`](https://furlan-lab.github.io/viewmastR/reference/viewmastR_infer.md)
  : ViewmastR inference
- [`augment_data()`](https://furlan-lab.github.io/viewmastR/reference/augment_data.md)
  : Augment data
- [`splat_bulk_reference()`](https://furlan-lab.github.io/viewmastR/reference/splat_bulk_reference.md)
  : Bulk reference
- [`confusion_matrix()`](https://furlan-lab.github.io/viewmastR/reference/confusion_matrix.md)
  : Confusion matrix
- [`plot_training_data()`](https://furlan-lab.github.io/viewmastR/reference/plot_training_data.md)
  : Training data plot
- [`make3file()`](https://furlan-lab.github.io/viewmastR/reference/make3file.md)
  : Export a Seurat Object to 10X-Style Files
- [`deconvolve_bulk()`](https://furlan-lab.github.io/viewmastR/reference/deconvolve_bulk.md)
  : Deconvolve bulk RNA-seq data using single-cell signatures

## Additional functions

Helper functions to add or modify elements

  : Calculate feature dispersion in a cell_data_set object

  : Select features in a cell_data_set for dimensionality reduction
- [`common_features()`](https://furlan-lab.github.io/viewmastR/reference/common_features.md)
  : Finds common features in a list of single cell objects

  : Get Selected Features for Ordering
- [`plot_feature_dispersion()`](https://furlan-lab.github.io/viewmastR/reference/plot_feature_dispersion.md)
  : Plot Feature Dispersion
- [`set_selected_features()`](https://furlan-lab.github.io/viewmastR/reference/set_selected_features.md)
  : Set Selected Features for Ordering
- [`sparseRowVariances()`](https://furlan-lab.github.io/viewmastR/reference/sparseRowVariances.md)

- [`writeMMgz()`](https://furlan-lab.github.io/viewmastR/reference/writeMMgz.md)

- [`plot_deconvolution()`](https://furlan-lab.github.io/viewmastR/reference/plot_deconvolution.md)
